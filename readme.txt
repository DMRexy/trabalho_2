This is the second assignment for Data Structures and Algorithms, finished 12/14/2021. 
The theme is depth-first search in a binary tree. 

I choose, to demonstrate the theme, a program that transforms a list of names in a binary tree of characters, allowing
for search by name, or impression of all utilizing DFS postorder. It also includes a gamelet of king of the hill, where
every character will fight in turn for the spot of survivor. Data entry comes from the namelist.txt fire, currently filled with
163 names, some repeated. Comments are in portuguese to allow for easier grading and sharing.
